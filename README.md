# Sistema de Medición de Impedancia con ESP32

Este repositorio contiene el desarrollo de un sistema electrónico para la medición de impedancias de baja magnitud utilizando el método Kelvin de cuatro hilos. El proyecto integra el diseño del hardware, el firmware del microcontrolador y una aplicación móvil para la visualización de los datos.

El sistema utiliza un microcontrolador ESP32 para la adquisición de señales analógicas, procesamiento de datos y transmisión inalámbrica mediante WiFi hacia una aplicación desarrollada en Flutter.

## Estructura del repositorio

```
impedance-measurement-system
│
├── hardware
│   ├── proteus
│   └── easyeda
│
├── firmware
│
├── mobile_app
│
├── docs
│
└── media
```

**hardware:** archivos de simulación en Proteus y diseño de la PCB en EasyEDA.

**firmware:** código desarrollado para el microcontrolador ESP32.

**mobile_app:** aplicación móvil desarrollada en Flutter para la visualización de los datos.

**docs:** documentación del proyecto e informe técnico.

**media:** imágenes y videos de demostración del sistema.


## Demostración

En la carpeta `media` se incluye un video de prueba donde se muestra una implementación experimental del sistema y la visualización de datos en tiempo real en la aplicación móvil.
